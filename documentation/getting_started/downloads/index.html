Downloads

We have precompiled binaries for most platforms, or you can also build from source if you prefer.

The Gort CLI provides many useful features and has no other dependencies. You install Gort separately from any framework, which means you can use it to program Arduinos with the Firmata firmware also compatible with Cylon.js, Gobot, Artoo, & other frameworks.

OSX

If you are using Homebrew, you can install Gort easily by typing:

$ brew install hybridgroup/tools/gort

Otherwise, download using one of the following links:

gort_0.9.0_darwin_386.zip
gort_0.9.0_darwin_amd64.zip
