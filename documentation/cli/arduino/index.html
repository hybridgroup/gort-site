Arduino

gort [global options] command [optional arguments...]


Description

Install avrdude, and upload sketches to your Arduino

$ gort arduino
Usage:
gort arduino install                                   # installs avrdude to allow uploading of sketchesto     Arduino
gort arduino upload firmata [port]                    # uploads Firmata sketch to Arduino
gort arduino upload rapiro [port]                     # uploads Rapiro sketch to Arduino
gort arduino upload [custom-firmware-filename] [port] # uploads a custom sketch to Arduino
Notes:

To be able to run the upload command you must run gort arduino install first. The serial port must have the correct permissions for your user to read and write, or run it as root (sudo it).

$ gort arduino upload ttyACM0
