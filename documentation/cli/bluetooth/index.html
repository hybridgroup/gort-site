Bluetooth

gort [global options] command [command options] [arguments...]


Description

Pair, unpair & connect to bluetooth devices.

$ gort bluetooth 
Usage:
gort bluetooth pair <address> [hciX]
gort bluetooth unpair <address> [hciX]
gort bluetooth unpair <address> [hciX]
